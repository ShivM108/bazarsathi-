## How to Build & Run the Project

Follow these steps to view and develop BazarSathi locally.

### **1. Clone the Repository**

Open your terminal or VS Code and run:

```
git clone https://github.com/ShivM108/bazarsathi.git
```

### **2. Open the Project Folder**

```
cd bazarsathi
```

### **3. Open `index.html` in Browser**

You can run it in two ways:

#### Method A – Double‑Click

Go to your project folder → double‑click **index.html**

#### Method B – VS Code Live Server (recommended)

* Install **Live Server** extension
* Right‑click `index.html`
* Click **Open with Live Server**

This automatically reloads your project whenever you make changes.

## Deployment Guide

You can host BazarSathi for free using **GitHub Pages**.

### Steps to Deploy:

1. Open your repository on GitHub
2. Go to **Settings**
3. Click **Pages** (left sidebar)
4. Under **Source**, select: `main` branch → `/root`
5. Click **Save`

Your site goes live at:

```
https://shivm108.github.io/bazarsathi/
```
